# ./tools/

In dem Ordner sind command line scripts um das Projekt mit gcc/g++/clang über die commandline zu compilen.

_compile.bat_ für windows (ungetestet, sollte aber funktionieren)
_compile.sh_ für unix

Bevorzugt wird jedoch die methode mit *make*
Der Commandlinebefehl funktioniert auf unix native.
Windows benötigt dafür [MSYS](http://www.mingw.org/wiki/MSYS) oder [CYGWIN](http://www.cygwin.com)
